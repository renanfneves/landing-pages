# Landing Pages

Standalone Next.js project.

## Scripts

- `npm run dev` — start development server
- `npm run build` — production build
- `npm run start` — run production server

## Project structure

- `app/` — Next.js App Router pages and layouts
- `components/` — page and section components
- `libs/ui-library/src/` — shared UI primitives used by the app
- `utils/` — project constants and helper values

## Install

```bash
npm install
```

## Build check

```bash
npm run build
```
